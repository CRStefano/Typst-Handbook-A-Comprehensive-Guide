# Typst: From Zero to Hero

> The handbook I wish I had when I started learning Typst.

Typst is probably the most exciting document preparation system created in decades.

It combines the power of LaTeX, the simplicity of Markdown, and the speed modern tools should have had from the beginning.

Yet there is a problem.

Most tutorials are either:

* Too short.
* Too fragmented.
* Too focused on a single topic.
* Written for people who already know half of the ecosystem.

This project was born to solve that.

**Typst: From Zero to Hero** is a complete handbook (for now in italian) that takes you from writing your first line of Typst to building academic papers, CVs, presentations, reusable templates, packages, automation workflows and production-ready documents.

The only prior knowledge required is being able to read.

---

## Why this handbook exists

I come from the academic world.

Like many people, I spent years fighting with Word, learning LaTeX, fixing broken packages, searching StackOverflow, and reading documentation scattered across dozens of websites.

I learned about typst through a my university professo. He immediately recognized the potential of this language. I've always used Latex, and he asked me to take a look at it so I could hold a small seminar on it someday.

When I discovered Typst, I immediately understood how beautiful it is.

The problem was that learning resources were still fragmented.

So I decided to write the resource I wanted to have:

**one document, one path, from beginner to advanced user.**

And much more.

The project is actively maintained and expanded.

The goal is simple:

> Finish the handbook and be able to build real Typst projects.

## Philosophy

This handbook follows a simple philosophy:

1. Learn by building.
2. Explain concepts from first principles using example.
3. Avoid unnecessary jargon.
4. Show complete code.
